# CT-001 – Validar login com campo senha em branco

## Pré-condições
- Usuário não autenticado
- Sistema disponível

## Passos para execução
1. Acessar o Portal do Servidor
2. Informar matrícula válida
3. Deixar o campo senha em branco
4. Clicar em Entrar

## Resultado esperado
- Acesso negado
- Mensagem informando que o campo senha é obrigatório

## Resultado obtido
- Mensagem genérica exibida
