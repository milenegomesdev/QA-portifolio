# Projeto de Quality Assurance – Portal do Servidor

Este repositório contém a documentação de testes funcionais e de usabilidade
realizados em um sistema web real, simulando a atuação de um QA Júnior
em um ambiente ágil.

#Objetivo
Validar o fluxo de login do Portal do Servidor, garantindo clareza nas mensagens
de erro e boa experiência do usuário.

#Ferramentas utilizadas
- Jira
- Navegador Web
- GitHub

#Escopo dos testes
- Testes funcionais de login
- Testes negativos (campos obrigatórios)
- Avaliação de usabilidade

#Casos de Teste
- CT-001 – Validar login com campo senha em branco

#Bugs encontrados
- BUG-001 – Mensagem genérica ao tentar login com campo obrigatório em branco

#Metodologia
- Testes manuais
- Sprint simulada
- Registro de bugs e rastreabilidade

#Conclusão
Foram identificadas falhas de usabilidade relacionadas à clareza das mensagens
de erro, impactando a experiência do usuário.
