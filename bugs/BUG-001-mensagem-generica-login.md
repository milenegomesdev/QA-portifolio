# BUG-001 – Mensagem genérica ao tentar login com campo obrigatório em branco

## Descrição
O sistema exibe uma mensagem genérica ao tentar login sem preencher o campo senha,
não informando claramente o erro ao usuário.

## Passos para reprodução
1. Acessar o Portal do Servidor
2. Informar matrícula válida
3. Deixar o campo senha em branco
4. Clicar em Entrar

## Resultado esperado
Mensagem clara informando campo obrigatório

## Resultado obtido
Mensagem genérica

## Severidade
Baixa

## Prioridade
Média
